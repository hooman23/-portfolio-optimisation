## What it does

- Finds the best mix of stocks for maximum return with minimum risk
- Draws the Efficient Frontier curve
- Finds Maximum Sharpe Ratio portfolio
- Finds Minimum Variance portfolio
- Uses Black-Litterman to blend market data with investor views
- Simulates 3,000 random portfolios with Monte Carlo
